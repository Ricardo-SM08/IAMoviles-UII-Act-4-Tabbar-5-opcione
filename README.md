## PROMPT
"Actúa como un desarrollador Senior de Flutter. Necesito que programes la estructura de navegación principal para mi aplicación de negocio automotriz llamada 'MySelfCar'. Debes entregarme el código completo de 7 archivos individuales listos para copiar y pegar.

1. Arquitectura y Archivo Main:

Crea el archivo main.dart.

Configura las rutas nombradas (Named Routes) en el MaterialApp.

La ruta inicial / debe cargar el controlador del TabBar.

Los otros 5 archivos deben estar dentro de una carpeta llamada exactamente mis_paginas_tab.

2. Navegación (TabBar Inferior):

Crea un archivo llamado pantalla_tabs.dart que contenga un DefaultTabController.

El menú de navegación debe ser un TabBar ubicado en la parte INFERIOR de la pantalla (bottomNavigationBar), NO en el AppBar.

Usa la siguiente paleta de colores de alta fidelidad: AppBar fondo Lila (#DCD0FF) con texto en Morado Oscuro (#4A148C) en tipografía cursiva. El fondo del TabBar inferior debe ser Morado Oscuro, con el indicador de selección y el texto activo en un Magenta Neón brillante (#E040FB).

3. Pantallas (Tabs):
Necesito 5 pantallas independientes que representen las áreas de mi negocio. Cada una en su propio archivo dentro de la carpeta mis_paginas_tab.

empleados_tab.dart (Icono: badge / Identificación)

clientes_tab.dart (Icono: people / Personas)

articulos_tab.dart (Icono: local_mall / Bolsa de compras)

refacciones_tab.dart (Icono: build / Herramienta)

carros_tab.dart (Icono: directions_car / Auto)

4. Diseño interno de cada pantalla:

Cada pantalla debe tener un Scaffold con fondo blanco y un SingleChildScrollView.

En la parte superior, un título centrado en Morado Oscuro.

Debajo del título, inserta una imagen usando Image.network. Usa URLs de prueba de alta calidad. Es OBLIGATORIO que la imagen esté dentro de un contenedor de 250px de altura, con bordes redondeados (radio de 20) y una sombra suave color magenta neón.

Para evitar que la imagen se deforme, se pixelee o se corte mal al cambiar el link por los míos de GitHub, debes usar obligatoriamente ClipRRect y la propiedad fit: BoxFit.cover en la imagen.

Debajo de la imagen, añade una breve descripción en texto gris centrada.
